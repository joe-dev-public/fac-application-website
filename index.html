Week 09: Build a feature

The design below represents a simple step sequencer, like you might find in a (software) drum machine, etc.

It won't make any sound ??(unless you click the button to load evil, derivative code)??, but is intended as an experiment in UI design.

There are a number of options for ways to interact with it, which are based on mouse input.

The purpose of this is to let you work out what combination of mouse inputs feels most fluid and natural for you.

Some of the controls present won't do anything, but it's important to have a bit of extra chrome thrown in so that the UI feels more like a real-world example.
